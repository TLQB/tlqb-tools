## INSTALLATION
Currently it is only supported Python3 onwards. It can be installed through pip:
```shell
pip install tlqb_tools
```

## USAGE
The following example illustrates the ease of use of this package:
```python
from tlqb_tools import tlqb
tlqb.sum_func(10,20)
```
